# Snake Game (Pygame)

Simple Snake game implemented with Pygame.

## Requirements

- Python 3.7+
- See `requirements.txt` for required packages. Install with:

```powershell
pip install -r requirements.txt
```

## Run

From the project root folder run:

```powershell
python snake_game_pygame.py
```

This will start the game window. Use the arrow keys to control the snake.

## Files

- `snake_game_pygame.py` — main game script.
- `requirements.txt` — Python dependencies.
